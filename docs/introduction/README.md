# NetCloth简介

```NetCloth```个人网络项目致力于让每个人都能打造属于自己的网络，从建立联系到形成复杂的网络生产关系，```NetCloth```将为所有人提供开放的基础设施和公开的区块链账本。

```NetCloth```区块链程序，包含: ```nchd``` 和 ```nchcli```

* ```nchd``` 是 ```NetCloth```全节点程序。
* ```nchcli``` 是命令行客户端，用来管理```key```并和全节点次交互。
