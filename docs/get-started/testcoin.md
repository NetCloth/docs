# 如何申领测试token

## 创建新账户

如果你已经有账户，可跳过此步骤。

如果没有账户，请点击[这里]()创建新帐户。

## 申请测试token

水龙头地址： ```https://docs.netcloth.org/nch/get_token?<address>```  
浏览器访问该地址，将```<address>```替换为你的钱包地址即可获得测试token

## 如何查看是否获得测试Token

方法1 通过区块浏览器查看

方法2 通过nchcli查看

使用下列命令进行查看，address为你的nch地址

```bash
nchcli query account [address]
```
